#Servitor Construction#

A "One Page WebApp"

This webapp is designed and coded by [Gemini Ai](https://wiki.dascent-sigils.com/index.php/Ai/Gemini) and serve the purpose of helping with construction of magical Servitors.

The page has editable textarea where user can input texts, upload an image (sigil) and generate a PDF file ready to download.

The drag-and-drop image upload is browser bases session, allows the PDF to grab the image but image itself is not store or hosted on a server, only in browser session which means, if you refresh the page, content is gone.
