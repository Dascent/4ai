<h2>1. Basic Sigil Generator</h2>
A "One Page WebApp" 

Allows user to create a sigil image based on distilation of the phrase they typed in.

The function is using a specific, yet classic [sigil distilation method](https://wiki.dascent-sigils.com/index.php/Sigilization_Methods)

<b>[Online Demo](https://dascent.github.io/4ai/source/sigil-generator/index.html)</b>


<h2>2. QR Sigil Generator</h2>
A "One Page WebApp" 

Allows user to create a QR Code sigil image based on user text input. More info on [wiki page about QR Sigil generator](https://wiki.dascent-sigils.com/index.php/QR_Code_Sigils)

<b>[Online Demo](https://dascent.github.io/4ai/source/sigil-generator/qr-sigil.html)</b>
